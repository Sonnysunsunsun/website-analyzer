üöÄ Getting Started

The SiteAnalyzer API provides RESTful endpoints for automated website analysis. Get comprehensive insights on SEO, performance, security, and more.

Authentication

All API requests require authentication using an API key in the header:

Authorization: Bearer YOUR_API_KEY

Base URL

https://website-analyzer-app.vercel.app/api/v1
Rate Limits:
  • Free tier: 10 requests/hour
  • Pro tier: 100 requests/hour
  • Enterprise: Custom limits

üìã Endpoints

Analyze Website

POST /analyze

Performs comprehensive analysis of a website including SEO, performance, security, and mobile optimization.

Request Parameters

Parameter Type Description Required
url string The website URL to analyze Required
mobile boolean Run mobile-specific tests (default: true) Optional
screenshot boolean Include screenshot in response (default: false) Optional
depth string Analysis depth: "basic" | "standard" | "comprehensive" Optional

Example Request

curl -X POST https://website-analyzer-app.vercel.app/api/v1/analyze \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com", "mobile": true, "depth": "comprehensive" }'

üîß SDK & Libraries

Node.js

npm install @siteanalyzer/sdk const SiteAnalyzer = require('@siteanalyzer/sdk'); const analyzer = new SiteAnalyzer('YOUR_API_KEY'); const result = await analyzer.analyze('https://example.com', { mobile: true, depth: 'comprehensive' });

Python

pip install siteanalyzer from siteanalyzer import SiteAnalyzer analyzer = SiteAnalyzer('YOUR_API_KEY') result = analyzer.analyze('https://example.com', mobile=True, depth='comprehensive' )

PHP

composer require siteanalyzer/sdk use SiteAnalyzer\Client; $analyzer = new Client('YOUR_API_KEY'); $result = $analyzer->analyze('https://example.com', [ 'mobile' => true, 'depth' => 'comprehensive' ]);
